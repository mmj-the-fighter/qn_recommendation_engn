# Question Recommendation Engine

This is the refererence Implementation of a recommendation engine for practice tests.  

## Where

* Low fail rate in a category -> High probability to ask that type of category next
* High fail rate in a category -> Low probability to ask that type of category next
* High number of questions asked in a category -> Reduced probability to ask that type of category next
